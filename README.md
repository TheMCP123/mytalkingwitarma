# Витёк Арматура

Загружай ZIP в Cloudflare Pages через Create deployment.

Переменные в Settings → Variables and secrets → Production:

- `DEEPSEEK_API_KEY` — твой ключ DeepSeek, лучше как Secret.
- `DEEPSEEK_MODEL` — опционально, например `deepseek-v4-flash`.

После добавления переменных сделай Redeploy.
